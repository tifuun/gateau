Generate SZ maps on-sky over a range of frequencies.

Parameters
SZsourceDictAn SZsource dictionary.
convolve_beamConvolve generated with diffraction-limited beams at specified frequencies. If True, need to specifiy a telescopeDict in order to get Dtel.
telescopeDictA telescope dictionary. Need not be given if convolve_beam is False.
ret_unitReturn maps in this unit. Choose between standard W m**-2 Hz**-1 sr**-1 ('SI'), MJy sr**-1 ('MJy') or mK ('mK').
Returns
SZ Datacube containing SZ intensity sky maps across the frequency range.
CMB Cosmic Microwave Background. Can be added to SZ map.
Az Array of Azimuthal co-ordinates.
El Array of Elevation co-ordinates.
